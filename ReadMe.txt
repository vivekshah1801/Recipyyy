Project Desc:
This is a Recipe portal. Where user can share their recipes. It has following features:

- Recipe Upload
- View Others recipe
- Share/Email recipe to others
- Cuisine Management
- Comments on Recipes
- Chef Management
- Chef Profile
- Recipe Description Page
- Admin Functions
- CRUD for recipes, cuisines, chefs
- Search Live on full site
- Image management
